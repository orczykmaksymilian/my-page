# Portfolio

Folder ZIP zawiera portfolio projektowe z zakresu **Systemów Informacji Przestrzennej**.  
Prezentuje wybrane realizacje z obszaru WebGIS, baz danych oraz systemów mobilnego GIS.

---

## Struktura projektu

- `index.html` – główny plik strony portfolio  
- `style.css` – arkusz stylów odpowiedzialny za wygląd strony  
- `images/` – folder zawierający wszystkie zdjęcia użyte w projektach (podzielone na podfoldery dla każdego projektu)  
- skrypty JavaScript – zawarte bezpośrednio w pliku `index.html` (obsługa modali i sliderów zdjęć)

---

## Jak uruchomić projekt

Aby uruchomić portfolio lokalnie należy otwórzyć plik `index.html`. Strona automatycznie uruchomi się w przeglądarce internetowej 

---

## Funkcjonalności

- interaktywna prezentacja projektów GIS
- okna modalne z opisami projektów
- slider zdjęć dla każdego projektu
- responsywny design (desktop + mobile)
- podział na sekcje: projekty, umiejętności, o mnie, kontakt
